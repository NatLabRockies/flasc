____ model_fitting ____

This contains methods for calibrating FLORIS models to SCADA (model_fit.py). The ModelFit approach
supersedes the previous beta version provided in floris_tuning.py. The `floris_tuning.py` module is
deprecated and will be removed in a future release. In accordance with semantic versioning, it will
remain available until the next major version bump; please migrate to `model_fit.py`.
